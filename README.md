# Reelax Assignment

A responsive React + Tailwind CSS implementation of the provided Reelax design.

## Tech Stack

- React JS
- Vite
- Tailwind CSS
- React Icons

## Features

- Responsive design
- Reusable components
- Billing form UI
- Order summary UI
- Mobile responsive layout

## Run Locally

```bash
npm install
npm run dev
```
